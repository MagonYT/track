# Track Phone Numbers 👀
**A simple project for tracking the phone number country and service 💻**
# How to use it? 🤔
- **``pip install phonenumbers colorama``**
- **``python track.py``**
- **Input the phone number with the country code**
- **Example (Applies to any area code): +1 123-456-7890**
